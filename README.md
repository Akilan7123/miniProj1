# 🔐 Password Generator Web App

## 📌 Objective
A simple Password Generator built using HTML, CSS, and JavaScript to create secure passwords.

---

## 🚀 Features
- Generate random passwords
- Select password length
- Include:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Copy password to clipboard

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

---

## 📂 Project Structure

Password-Generator/ │── index.html │── style.css │── script.js │── README.md

---

## ▶️ How to Run
1. Clone or download the project
2. Open `index.html` in your browser
